## Ключевые тренды

### 1. Claude Code доминирует
80% всех outliers (12 из 15) связаны с Claude Code. Это явный тренд — тема Agent Teams и Claude Code привлекает аномально высокий интерес аудитории. Видео "Claude Code's Agent Teams Are Insane" набрало 1093x от среднего канала — экстремальный outlier.

### 2. RU-рынок активно подхватывает тренд
3 видео на RU-рынке с высокими outlier scores (до 135x) показывают, что русскоязычные авторы быстро реагируют на тренд Agent Teams. Тема актуальна для обоих рынков.

### 3. "Анти-agentный" контент тоже работает
Видео "STOP Building AI Agents" (41.89x, 149K views) показывает спрос на контент с contrarian-позицией. Аудитория ищет баланс между хайпом и практичностью.

### 4. Длинноформатные курсы набирают
"AI Agents Full Course" (9.56x, 94K views, 2 часа) и "Claude Code Full Tutorial" (6.53x, 385K views) — аудитория готова смотреть длинный образовательный контент по теме.

## Рекомендации

1. **Claude Code Agent Teams** — горячая тема. Создавать контент именно про Agent Teams, pipeline агентов, практические кейсы.
2. **Contrarian angle** работает — формат "не делайте X, делайте Y" привлекает внимание.
3. **Туториалы и курсы** — длинный формат не отпугивает, а привлекает целевую аудиторию.
4. **RU-рынок** — ниша для русскоязычного контента про Claude Code/Agent Teams ещё не перенасыщена.
5. **Timing критичен** — все топ-outliers опубликованы в последние 2-3 недели, быстрая реакция на новые фичи = высокие scores.
